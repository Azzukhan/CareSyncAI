# CareSync

CareSync is split into two top-level apps:

- `frontend/` for the React, TypeScript, Vite client
- `backend/` for the FastAPI service

## Frontend

```sh
cd frontend
cp .env.example .env
npm install
npm run dev
```

Frontend-specific notes live in `frontend/README.md`.

## Backend

Backend setup and runtime instructions live in `backend/README.md`.
